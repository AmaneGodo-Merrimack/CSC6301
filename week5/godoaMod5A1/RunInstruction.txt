# Run Instructions

## Compile

```bash
javac *.java
```

## Run

```bash
java Demo
```

## Generate JavaDocs

```bash
javadoc -d doc *.java
```

## Files Included

* NotificationMedium.java
* EmailService.java
* SMSService.java
* WhatsAppService.java
* AlertSystem.java
* Demo.java
* godoaMod5A_UMLDesign.pdf
* doc/ (generated JavaDocs)
